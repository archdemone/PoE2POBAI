Core Question: What should clone/diff enable?

A

Clone → variant build

Copy a snapshot to a new ID with an optional label. Then the LLM can propose changes: "if you swap this ring for a Ming's Heart, your chaos res drops but DPS goes up." The variant is a full standalone snapshot.

B

Diff two snapshots

Compare any two builds side by side: items changed, skill swaps, passive tree differences, defense deltas. Useful for "show me what changed between my last two imports."

C

What-if: apply & compare

LLM proposes a change set (e.g. swap ring, allocate 5 nodes, change gem). System applies it to a clone, computes diff, and shows before/after stats. The most advanced option.

D

Expose as MCP tools

Add clone_build, diff_builds, patch_build tools so any MCP client (Claude Desktop, cursor, etc.) can use these, not just the web UI.
