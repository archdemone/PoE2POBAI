Text streamingStep-by-step only
ComponentsFull tree + tests

Data Flow

1. Browser opens WebSocket to pobai-server
2. User sends message via WS
3. Server forwards to LLM (OpenRouter)
4. LLM returns tool_calls → server → WS → browser shows "Running X..."
5. Browser calls HTTP API → executes tool → sends results via WS → server → LLM
6. Repeat until LLM returns text → browser shows final response
