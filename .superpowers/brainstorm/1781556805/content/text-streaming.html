Step 4 of 5: Text Streaming

When the LLM generates the final response text, how should it appear?

A

Token-by-token streaming

Text appears character by character like ChatGPT. Feels responsive, but requires the LLM provider to support streaming and adds WS message overhead.

B

Step-by-step (no text streaming)

Show tool execution progress in real-time, then display the full LLM response at once. Simpler, less WS traffic, still feels interactive because tool steps fill the gap.
