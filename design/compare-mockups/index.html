Compare section — redesign mockups

Same data (Dconnic_Monka Lv 91 vs Target Guide Lv 96). Click a direction to view it full-size.

DIRECTION 1
Side-by-side + delta gutter
Two build columns with a center strip showing the difference. Mirrors PoB's own layout — instantly tells you which side has more.
DIRECTION 2
Swap checklist most actionable
Turns the diff into a to-do list: Add / Remove / Tweak, with checkboxes and a progress bar. Best for actually copying a build step by step.
DIRECTION 3
Dashboard + scoreboard
Top scoreboard of totals, then compact category cards (defenses, offense, skills as chips). Best at-a-glance overview.
DIRECTION 4
Visual bars + skill matrix
Stats as paired bars (bigger bar wins), skills as a left/center/right matrix. Most visual — no reading numbers required.

Hybrid + full character sheet

Scoreboard + complete PoE2 stat panel (STR/DEX/INT, resistances, defences, offence) + swap checklist — four UI treatments of the same idea.

HYBRID · BASE
Stat rows + checklist recommended
Full character sheet as side-by-side rows with bars and a Δ column; winning side highlighted. Swap checklist on the right.
HYBRID · A
Tabbed dense grid
Two-stats-per-row compact grid so the whole sheet fits without scrolling. Tabs for Sheet / Skills / Passives. Left-border colour shows who leads.
HYBRID · B
Diverging tug-of-war bars
Each stat is one bar growing out from a centre line — mine left, target right. The dominant side is obvious at a glance.
HYBRID · C
Tile dashboard (PoE skin)
Stat groups as gold-trimmed cards with "mine / target / tied" verdicts. Warmer in-game aesthetic.
